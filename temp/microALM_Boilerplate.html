1. Philosophy & Usage

Welcome to microALM. This is a single-file Application Lifecycle Management (ALM) solution.

  • Self-Contained: Requirements, Specs, and Tests in one HTML file.
  • Traceable: Clickable links between items (Req ↔ Spec ↔ Test).
  • Editable: Edit content directly in the browser and save the file.
  • Integrity-Aware: Calculates its own hash (UIDD) for version verification.
  • Mobile-Ready: Supports horizontal scrolling for complex tables on small screens.
