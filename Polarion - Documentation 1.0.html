AI Doc Standard 1.4 - High Contrast Spec

Status: ACTIVE | Version: 1.4
Focus: Traceability & Self-Documentation

1. Philosophy & Usage

This document acts as the Meta-Specification. It is a self-contained, living document that defines its own structure and behavior. It integrates Requirements, Specifications, and Tests in a single file to ensure complete traceability.

Core Concepts:

  • Single Source of Truth: No external dependencies (CSS/JS embedded).
  • Living Document: Edit, Extend, and Save directly within the browser.
  • Traceability: Every requirement is linked to a specification and verified by a test case.
