microALM: PnL Tracker CSV Parser

Status: ACTIVE | Version: 3.3
Focus: CapTrader CSV to XML (Mobile Ready)

1. Philosophy & Usage

This document serves as the migrated documentation for the PnL Tracker 3.0 (CSV Importer) in the microALM format. Its primary goal is to define the logic for parsing CapTrader/IBKR CSV files and converting them into a structured XML format (Single Source of Truth).

Core Concepts:

  • Single Source of Truth: The parser converts raw CSV data into a standardized trades.xml file.
  • Auto-Integrity: The document calculates its own SHA-256 hash to ensure the integrity of the requirements.
  • Living Document: Requirements, specifications, and test cases can be edited directly within the browser.

4. Specifications (The "How")

ID Module Title Technical Implementation Traces To Covered By
